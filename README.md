# Jay German's Homebrew Tap

A [Homebrew](https://brew.sh/) tap for tools maintained by Jay German.

## Usage

```sh
brew tap jsgermanaai/tap
brew install jsgermanaai/tap/mlx-training-studio
```

## Formulae

| Formula | Description |
|---|---|
| `mlx-training-studio` | Installer CLI for [MLX Training Studio](https://github.com/jsgermanaai/mlx-training-studio-installer) — a native macOS GUI for fine-tuning LLMs with mlx-lm-lora. Installs the CLI; run `mlx-training-studio install` to build and install the app. |

## More information

See the [installer repo](https://github.com/jsgermanaai/mlx-training-studio-installer) for full documentation.
